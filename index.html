Creates a bunch of rows and then recursively divides them into chunks and then chunks of chunks and so on until the number of elements per "leaf chunk" is that of the value specified below. Each chunk has content-visibility:auto so there's no layout/rendering cost for the vast majority (99.99...%) of leaf rows. Should really only be limited by the memory required to store the divs. See readme for more info.
